# Instructions

1. Download SSP_v1.0.0.5.exe.zip
2. Extract the files on your desktop.
3. Right click on SSP_v1.0.0.5.exe and press Run as administrator
4. Choose any/all of the options you'd like
5. You can read the output in the console dialog to see if it worked.
    That's it!
